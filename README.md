## 🛠️ Como foi desenvolvido

Criei meu portfólio do zero usando **HTML e CSS puro**, sem nenhum framework ou template pronto!

Comecei pela estrutura básica do HTML — o esqueleto da página com as tags `h1`, `p`, `ul` e `section`. Depois criei um arquivo `style.css` separado e fui conectando os dois com a tag `link`.

No CSS, aprendi a usar:
- Cores em hexadecimal para o fundo e os textos
- Google Fonts para importar fontes mais bonitas
- Flexbox para organizar o layout lado a lado
- Gradientes para deixar as cores mais vivas
- Animações com `@keyframes` para os blobs do fundo e a borda da foto

Fui construindo aos poucos e salvando cada melhoria no GitHub com commits descritivos, para manter o histórico da evolução do projeto.

Todo o processo foi feito no VS Code, usando a extensão Live Server para visualizar as mudanças em tempo real no navegador.
